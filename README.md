### 🌟 Featured Projects

#### 🔬 [NGS-RAG Assistant](https://github.com/AndErem314/ngs-rag-assistant)
Privacy-first RAG system for querying scientific protocols with natural language. Full pipeline: PDF extraction → chunking → embeddings → vector search → LLM generation with source citations. Runs fully locally via Ollama — no data leaves your machine.

#### 📈 [BacktestBot](https://github.com/AndErem314/BacktestBot)
Python-based multi-asset backtesting platform with Ichimoku + PSAR + MACD strategies, LLM-powered strategy optimization, and automated PDF reporting. Modular ABC architecture supporting both crypto and commodity markets.

#### 🌊 [Ichimoku Assistant](https://github.com/AndErem314/Ichimoku_Assistant)
Real-time crypto trading signal monitor using Ichimoku Cloud analysis. AI-enhanced insights via LLM, scheduled 4-hour checks, and notifications via Discord & Telegram. Docker-ready.

#### 🎬 [MoviWebApp](https://github.com/AndErem314/MoviWebApp)
Flask web application with user authentication, RESTful APIs, and OMDb integration. Clean OOP architecture with SQLAlchemy + PostgreSQL backend.

#### 🗣️ [Local Voice Language Tutors](https://github.com/AndErem314/local-voice-spanish-tutor)
Voice-powered language tutors running fully local — Ollama for LLM, Moonshine for STT, Kokoro for TTS. No cloud APIs, no data sent anywhere. Available for [Spanish](https://github.com/AndErem314/local-voice-spanish-tutor) and [Portuguese](https://github.com/AndErem314/local-voice-portuguese-tutor).

#### 💱 [SOL DCA Bot](https://github.com/AndErem314/sol-dca-bot)
Dollar-cost averaging trading bot for Solana via Jupiter Exchange with Phantom wallet integration. *Actively maintained — more strategies coming soon.*
